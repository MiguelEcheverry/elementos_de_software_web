1. Conceptos básicos de Internet

a. Servidor DNS**
Es un sistema que traduce nombres de dominio en direcciones IP, permitiendo que los navegadores encuentren los servidores correctos en Internet.

b. IP (Dirección IP)**
Es un identificador único asignado a cada dispositivo conectado a una red. Funciona como una “dirección” para localizar equipos en Internet.

c. Protocolo**
Es un conjunto de reglas que permiten la comunicación entre dispositivos en una red. Ejemplo: HTTP, HTTPS, FTP.

d. Nombre de dominio**
Es el nombre fácil de recordar que se usa para acceder a un sitio web (por ejemplo, www.ejemplo.com) en lugar de usar su dirección IP.

e. Directorio y nombre de archivo**
Es la ruta dentro de un servidor web que indica la ubicación exacta de un recurso.


2. Roles en el desarrollo web

a. Diseñador UI/UX**
Se encarga de diseñar la apariencia y la experiencia de usuario de un sitio o aplicación, asegurando que sea atractiva, intuitiva y fácil de usar.

b. Cloud Manager / Ingeniero de infraestructura en la nube
Administra y mantiene los servicios en la nube (como servidores, almacenamiento y redes), garantizando disponibilidad, seguridad y escalabilidad.

c. Administrador de base de datos (DBA)
Gestiona las bases de datos, asegurando su correcto funcionamiento, seguridad, respaldo y optimización del rendimiento.

d. Project Manager / Director del Proyecto
Planifica, organiza y supervisa el desarrollo del proyecto, coordinando al equipo para cumplir objetivos, tiempos y presupuesto.



3. ¿Qué significa que un lenguaje sea Turing completo?

Un lenguaje es Turing completo cuando puede resolver cualquier problema computacional, siempre que tenga suficiente tiempo y memoria. Es decir, puede ejecutar cualquier algoritmo.



4. ¿Por qué CSS y HTML no son lenguajes de programación?

Porque no permiten realizar lógica compleja ni control de flujo (como condiciones o bucles).

HTML estructura el contenido.
CSS define el estilo.
 Ninguno puede ejecutar algoritmos por sí mismo.



5. ¿Qué es una etiqueta en HTML?

Una etiqueta en HTML es un elemento que se utiliza para definir la estructura y contenido de una página web. Generalmente tiene una etiqueta de apertura y una de cierre.
Ejemplo: `<p>santy_nos_dijo_esto</p>`
